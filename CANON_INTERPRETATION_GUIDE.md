Intake|Boundary — Canon Operational Interpretation
Version: 1.0
________________________________________
Interpretation (Prohibited)
Any behavior that:
• Infers importance
• Ranks outputs
• Suppresses fields
• Repairs malformed input semantically
• Applies cross-field reasoning
• Adjusts output based on context
Is prohibited.
________________________________________
Authority
Authority is defined by:
• Canon document
• Execution contract
• Versioned configuration
Human role is limited to providing explicit inputs.
There is no discretionary override.
________________________________________
Determinism
Determinism requires:
• Identical inputs
• Identical configuration
• Identical versions
Produce identical outputs.
Time may be recorded but must not influence execution.
